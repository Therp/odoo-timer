# Privacy Policy – Therp Timer

Therp Timer does not collect, store, or transmit personal user data to external servers.

## Data Usage

The extension operates locally within the user's browser and communicates only with user-configured Odoo servers.

### Local Storage

The extension stores configuration data such as:

* Odoo host URL
* Selected database
* Timer state
* User preferences

This data is stored locally in the browser and is not shared externally.

### Authentication

The extension uses existing Odoo session cookies to authenticate API requests to the user's own Odoo instance.

No authentication data is collected, stored, or transmitted to third-party services.

### External Communication

The extension communicates only with:

* User-configured Odoo servers

No data is sent to any third-party or developer-controlled servers.

### Notifications

Notifications are used solely to inform users about timer-related events and do not contain personal data.

## Data Sharing

Therp Timer:

* Does not sell user data
* Does not share user data with third parties
* Does not track users across websites

## Security

All operations are performed locally or directly between the user and their Odoo server.

## Changes

This privacy policy may be updated in future versions.
