Description

This is a cross-plaform Sunflower Timer works in Firefox and Chrome browsers with the aim of registering working hours of individual issues based on Odoo timesheets.

Features

  • Start issue timer.
  • Edit issue time
  • Stop issue timer
  • Create odoo timesheet line record linked to analytic account.
  • Change issue state - eg. start or finish issue
  • Support tasks instead of issues
  • Show own issues or all issues
  • Add a Remote host
